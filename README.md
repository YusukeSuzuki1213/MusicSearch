# MusicSearch
iTunesから音楽を検索できるデスクトップアプリケーションです。  
iTunes Search API を使用することにより楽曲サンプル、ジャケット、曲名、アーティスト名、アルバムシングル名を取得しました。  
Electronを使用し、JavaScript、HTML、CSSのみでデスクトップアプリケーションを作りました。
